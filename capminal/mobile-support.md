# Mobile Support

User can install Capminal.ai mobile app following below steps:

1\. Open [Capminal](https://www.capminal.ai/) on mobile browser (Safari/Chrome...)

2\. Click on '**Share**' button.

3 . Then click on '**Add to Home Screen**' button.
